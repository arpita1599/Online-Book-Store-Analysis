# Online Book Store Data Analysis (SQL Project)

## Project Overview
This project analyzes an Online Book Store database using SQL. The dataset contains information about books, customers, and orders. The objective of this project is to perform data analysis using SQL queries to extract meaningful insights such as sales performance, customer activity, and inventory status.

## Dataset
The project uses three CSV files that represent relational tables in a database.

- *Books.csv* – Contains information about books such as title, author, genre, price, and stock.
- *Customers.csv* – Contains customer details including name, city, and country.
- *Orders.csv* – Contains order transaction details such as order date, quantity, and total amount.

These tables are connected using common columns:
- Book_ID
- Customer_ID

## Project Objectives
- Analyze book sales data
- Identify popular genres and best-selling books
- Understand customer purchasing behavior
- Calculate total revenue generated from orders
- Monitor inventory stock levels

## SQL Analysis

### Basic Queries
- Retrieve all books in the Fiction genre
- Find books published after 1950
- List customers from Canada
- Show orders placed in November 2023
- Calculate the total stock available
- Find the most expensive book
- Retrieve orders where total amount exceeds $20
- Identify books with the lowest stock

### Advanced Queries
- Total number of books sold by each genre
- Average price of books in the Fantasy genre
- Customers who placed at least two orders
- Most frequently ordered book
- Top 3 most expensive Fantasy books
- Total quantity of books sold by each author
- Customer who spent the most on orders
- Remaining stock after fulfilling all orders
